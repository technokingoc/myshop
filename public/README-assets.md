# Required Assets for SEO & Performance

The following image assets need to be created for optimal SEO and performance:

## Open Graph Images
- `og-image.png` (1200x630px) - Main OG image for social sharing
- `og-image-square.png` (600x600px) - Square version for some platforms

## App Icons (for manifest.json)
- `icon-192x192.png` - 192x192px app icon
- `icon-512x512.png` - 512x512px app icon
- `favicon.ico` - Standard favicon
- `apple-touch-icon.png` - 180x180px for iOS

## Screenshots (for app listing)
- `screenshot1.png` - Desktop view (1280x720px)
- `screenshot2.png` - Mobile view (750x1334px)

## Brand Colors
- Primary: #22c55e (Green)
- Background: #ffffff (White)
- Text: #0f172a (Dark slate)

## Design Guidelines
- Include "MyShop" logo/text
- Use the green theme color (#22c55e)
- Show storefront/commerce elements
- Keep it clean and professional
- Include tagline: "Create Your Online Store in Minutes"

These should be created by a designer or using tools like Figma, Canva, or similar.
